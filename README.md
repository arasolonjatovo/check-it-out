## Lancer le projet en local

Cloner le projet

```bash
  git clone git@github.com:arasolonjatovo/check-it-out.git
```

Se rendre dans le dossier du projet

```bash
  cd check-it-out
```

Installer les dépendances

```bash
  npm install
```

Démarrer le serveur

```bash
  npm run start
```
